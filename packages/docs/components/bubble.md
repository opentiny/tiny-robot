## 角色

默认角色为 ai，设置成 user 可以调换头像和内容的位置

<div style="background: #f9f9f9; padding: 24px; border-radius: 8px">
  <TinyBubble role="user" content="TinyVue 是一款由华为开源的轻量级 Vue.js 组件库，专为企业级应用设计。"/>
</div>
